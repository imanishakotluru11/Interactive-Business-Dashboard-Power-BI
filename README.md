Overview
This project showcases an interactive Business Intelligence dashboard built using Microsoft Power BI. The dashboard transforms raw data into meaningful insights using data cleaning, modeling, and visualization techniques.
Objective
To analyze business data and present key performance indicators (KPIs) in a clear, interactive format for better decision-making.
Tools Used
Microsoft Power BI
Power Query
DAX
Data Modeling
Key Features
Interactive dashboard with slicers
KPI cards and performance metrics
Bar, line, and pie chart visualizations
Clean and user-friendly layout
